# Advanced Face Emotion Detection System

## 📋 Overview
A production-level face emotion detection system with real-time processing, analytics dashboard, and comprehensive logging capabilities.

## ✨ Features
- Real-time webcam emotion detection
- Multi-face detection and tracking
- 7 emotion classifications (Happy, Sad, Angry, Surprise, Fear, Disgust, Neutral)
- Interactive Streamlit dashboard
- Emotion analytics and visualization
- Automatic logging to CSV/SQLite
- Image upload support
- Screenshot capture
- Voice feedback
- Emotion heatmaps
- Report generation

## 🛠️ Tech Stack
- **Python 3.8+**
- **OpenCV** - Face detection and image processing
- **TensorFlow/Keras** - Deep learning model
- **Streamlit** - Web interface
- **SQLite/CSV** - Data logging
- **Plotly/Matplotlib** - Visualization
- **Dlib** - Advanced face detection
